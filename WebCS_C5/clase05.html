Unidad 3: Programación Frontend con JS

Curso básico de ambiente web

Introducción a JavaScript

JavaScript es un lenguaje de programación que permite crear páginas web interactivas y dinámicas.

Variables y Tipos de Datos

En JavaScript, puedes declarar variables usando let, const o var.

let nombre = "Juan";
             const edad = 30;
             var ciudad = "San José";

Condicionales

Los condicionales permiten ejecutar código basado en condiciones.

if (edad >= 18) {
             console.log("Eres mayor de edad");
             } else {
                 console.log("Eres menor de edad");
             }

Eventos

Los eventos permiten interactuar con el usuario. Por ejemplo, puedes capturar un clic en un botón.
